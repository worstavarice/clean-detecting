# clean-detecting
# утилита, которая позволит Вам просматривать:
Время включения ПК
Подключенные диски
Анализ журналов событий
Анализ .pf (Проверка reg.exe/fsutil.exe/wevtutil.exe/+ проверка readonly + hidden файлов)
Статус служб
Дата очистки корзины
Рецент (чистый или нет + дата изменения)
Регистрируется ли activitiescache.db
Проверка очистки shellbags
Проверка, есть ли consolehost_history и дата изменения
Время запуска текущего java-процесса 
Доступ к реестру, диспетчеру задач итд
Доступ к префетч

Также добавлена тестовая функция (2), которая детектит некоторые запущенные .py файлы и клинеры.
